how to change the file permissions (chmod)
symbolic method and nummeric method

Symbolic method ->
u -> user(owner)
g -> group
o -> others
a -> all

operators ->
+ -> add permissions
- -> subtract permissions
= -> assign permissions

mkdir script.sh
chmod g-x script.sh
chmod g+x script.sh

Nummeric method ->
read -> 4
write -> 2
exceute -> 1

rwx -> 7
rw- -> 6
r-x -> 5
r-- -> 4
-wx -> 3
-w- -> 2
--x -> 1
--- -> 0

chmod 755 -> user(rwx) group(r-x) others(r-x)

chmod -R for recursive permission change


users and groups -->
user -> single person using device
group -> collection of users
